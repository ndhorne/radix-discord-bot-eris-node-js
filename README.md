# radix-discord-bot-eris-node-js
Radix Discord Bot

A general-purpose Discord bot powered by JavaScript, Node.js, and the Eris Node.js Discord library featuring:

- Games
  - Tic Tac Toe
  - Connect 4
  - Hangman
  - Definitely Not Wordle
  - The Matchstick Game
  - Random Acts of ASCII
- Galleries
  - The Metropolitan Museum of Art
  - Bob the Angry Flower
  - Pokey the Penguin
  - xkcd
- MUDs
  - MUD Maze
- Misc
  - D20 Dice
  - Horoscopes
  - Fortunes
  - Magic 8 Ball
  - Currency Converter
  - ROT13 encoder/decoder
  - Random Password Generator

Both prefix ($rdx) and slash commands supported, `$rdx help` to get started
